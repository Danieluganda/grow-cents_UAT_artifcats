2. Assessment Workflow Lifecycle



1. Draft State: Data is editable; no maturity score calculated.
2. In Progress: 50%+ fields completed; preliminary HUD visibility.
3. Submitted: Locked for standard users; "Handshake" with High Command initiated.
4. Paid: Subscription verified; Full Strategic Intelligence unlocked.
5. Report Unlocked: PDF/Expert Consultation nodes active.

3. Post-Submission Edit Rules (Grace Period)



  • Window: 7 Days after initial submission.
  • Trigger: "Request Unlock" button in Club Console.
  • Node: Only editable by L4 (President) or above.
  • Logic: Any edit resets the `reporting_date` timestamp.
