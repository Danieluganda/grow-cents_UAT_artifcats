Growing Cents: ERD & Data Dictionary v1.0



Comprehensive relational architecture and schema definitions for the Growing Cents Institutional Intelligence System.

1. Entity Relationship Diagram (ERD)



Growing Cents ERD



1.1 Core Relationships


  • Users <-> Clubs: Multi-tenancy via `club_user` pivot. Multiple roles per user across different clubs.
  • Clubs <-> Assessments: One-to-Many. Each club maintains a history of governance audits.
  • Assessments <-> Questions: One-to-Many. Assessments are built from a centralized question bank.
  • Clubs <-> Subscriptions: One-to-One. Subscriptions gate access to "Elite" vs "Basic" logic nodes.




2. Data Dictionary (Institutional Core)



2.1 Table: `users`


Identity management and authentication node.
ColumnTypeNullableDescription

`id`BIGINT (PK)NoPrimary Key
`name`VARCHAR(255)NoFull legal name of the operative
`email`VARCHAR(255)NoUnique identifier (UK) for system access
`status`TINYINTNo1: Active, 0: Suspended
`last_login_at`TIMESTAMPYesLast successful 2FA handshake
`two_factor_secret`TEXTYesEncrypted 2FA seed

2.2 Table: `clubs`


Institutional governance units.
ColumnTypeNullableDescription

`id`BIGINT (PK)NoPrimary Key

2.3 Table: `assessments`


Governance audit and maturity tracking.
ColumnTypeNullableDescription
