Internal Refund Logic



If a club requests a strategic retreat (refun), the following protocol is invoked:
1. Mark Subscription: `status = 'refunding'`
2. Revoke Elite Access: Downgrade logic at middleware layer.
3. Verify Yield Correction: Audit log entry for financial rollback.

Handling Logic



1. Validate Signature: Ensure payload originates from the verified yield stream.
2. Check SubscriptionID: Prevent double-entry in the ledger.
3. Update Institutional Status: Reactivate club features immediately.
4. Log to Audit Trail: Store yield success event.
