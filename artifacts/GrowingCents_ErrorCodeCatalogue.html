Error Code Catalogue

Complete reference of HTTP response codes returned by the Growing Cents platform. Every route in routes/web.php follows standard Laravel HTTP conventions. This catalogue maps each code to its trigger condition, which endpoint(s) can return it, and the correct handling approach.

1. Authentication & Session Errors (4xx)

CodeStatusWhen It OccursEndpointsHandling
401UnauthenticatedUser is not logged in and tries to access a protected route. Laravel Sanctum/session has expired.All routes behind auth middlewareLaravel auto-redirects to /login. On AJAX: returns JSON {"message":"Unauthenticated."}
403ForbiddenUser is logged in but does not have the required role or permission. E.g. Club Member attempting to POST /assessments.All routes behind role:* or permission:* middlewareReturns 403 page or JSON. Common cases: Club Member submit attempt, non-admin accessing /admin/*
404Not FoundResource does not exist or user is trying to access a club/assessment they do not belong to. findOrFail() throws this.All /{club}/assessments/{assessment} routesReturns 404 page. Prevents horizontal access — Club A cannot access Club B records even if they guess the ID.
419CSRF Token MismatchForm submitted without a valid CSRF token. Usually happens when a session expires mid-form, or on direct API calls without the token header.All POST/PUT/PATCH/DELETE routesLaravel returns 419. Fix: ensure @csrf in Blade forms. For AJAX: include X-CSRF-TOKEN header from meta[name=csrf-token].
422Unprocessable EntityRequest validation failed. Required fields missing, wrong format, or business rule violated.All POST/PUT/PATCH routes with validationReturns JSON with {"message":"...","errors":{"field":["reason"]}}. Common cases below.
429Too Many RequestsRate limit exceeded. OTP resend throttled at 3 attempts per minute. Login throttled at 5 attempts per minute.POST /login, GET /verify-2fa/resendReturns 429 with Retry-After header. User must wait before retrying.

3. Server Errors (5xx)

CodeStatusWhen It OccursHandling
500Internal Server ErrorUnhandled exception in application code. DB connection failure. Missing JSON schema file (complete_assessment_data.json not found). DomPDF render failure on malformed report data.Laravel logs to storage/logs/laravel.log. Returns error page in production (APP_DEBUG=false). During UAT: APP_DEBUG=true for full stack trace.
503Service UnavailableApplication is in maintenance mode (php artisan down). Hosting server overloaded.Laravel returns maintenance page. Run php artisan up to restore.
504Gateway TimeoutPDF generation taking too long for very complex reports. External mail service (Mailtrap/SMTP) timing out.Increase max_execution_time in PHP config. Check mail server connectivity.
