4. OTP / Email Testing

All emails (OTP codes, invitation emails, assessment reports) are captured in the shared Mailtrap inbox. When logging in as any test account, the 6-digit OTP will appear in the Mailtrap inbox. Mailtrap credentials will be shared separately once the inbox is configured.

5. Known Test Gotchas

  • Every login requires a 2FA OTP — check Mailtrap inbox for the code after entering email/password
  • OTP expires after 10 minutes — request a resend if it expires
  • Club Admin account is admin of Kampala Investment Club only — cannot access Entebbe data
  • Club Member account cannot submit assessments — any attempt returns 403
  • Payment gate is currently inactive — all completed reports are accessible without subscription during UAT
