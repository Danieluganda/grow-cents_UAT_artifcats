3. Login Flow — What to Expect

  1. Go to https://assessment.growing-cents.com
  2. Enter email + password
  3. System sends a 6-digit OTP to the email address
  4. Check the Mailtrap inbox for the OTP — it expires in 10 minutes
  5. Enter OTP → login completes
  6. Super Admin → redirected to admin dashboard
  7. Club user with 1 club → redirected directly to club dashboard
  8. Club user with 2+ clubs → shown club selector page
📌 OTP Note: If OTP expires before you enter it, click "Resend Code" on the verification page. Check Mailtrap again for the new code.

4. What the Super Admin Account Can Test

  • Admin dashboard — all clubs overview
  • User management — approve, suspend, edit users
  • Role management — create custom roles, assign permissions
  • Audit log — view all system actions
  • Assessment results across all clubs
  • System settings — maturity colours, nudge configuration
  • Log exports (CSV/XML)

5. What the Club User Account Can Test

  • Full registration and club creation flow
  • Complete an assessment end to end (Area 0 through Area 9)
  • Autosave and draft resume
  • Assessment submission and report viewing
  • PDF download and email report
  • Club dashboard — historical assessments, maturity score
  • Club settings update

6. Known Gotchas for Testing

  • Every login requires a 2FA OTP — always check Mailtrap inbox after entering credentials
  • OTP expires after 10 minutes — resend if needed
  • Payment gate is currently inactive — all completed reports are accessible without subscription during UAT
  • SSL must be confirmed live before any browser testing can begin — HTTP will not work
  • Club Member role cannot submit assessments — returns 403 Forbidden (this is correct behaviour per SRS Section 8)
